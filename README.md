# TargetFinder Demo

This repository contains a small demo website and runnable examples for target selection techniques developed around TargetFinder.

The main implementation is not copied here. This demo imports `target-finder-toolkit` from the main repository so that future improvements in TargetFinder can be reused automatically.

## Demo Website

The website is built with MkDocs and can be hosted on GitHub Pages.

## Techniques

- TargetFinder
- Bubble Cursor
- DynaSpot
- Semantic Pointing
- Ninja Cursors

## Supported Platforms

The toolkit includes platform-specific support for macOS, Windows, and Linux/X11.

Linux support targets X11. Wayland is not fully supported for cursor hiding and click redirection.
