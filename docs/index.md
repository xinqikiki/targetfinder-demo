# TargetFinder Demo

This website presents short demonstrations of several target selection techniques developed around TargetFinder.

The goal is to provide a simple public-facing demo page with videos, installation instructions, and runnable examples.

## What is shown here?

- Target detection with TargetFinder
- Bubble Cursor
- DynaSpot
- Semantic Pointing
- Ninja Cursors

## Link with TargetFinder

This demo project does not duplicate the source code of TargetFinder.

Instead, it explicitly imports `target-finder-toolkit` as a dependency. This keeps the demo connected to the main toolkit, so improvements in the main project can be reused by this demo repository.
